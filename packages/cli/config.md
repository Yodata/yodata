### yodata config <cmd>

```sh
$ yodata config <cmd>

view and manage yodata configuration

Commands:
  yodata config delete <key>       remove <key> from config store
  yodata config get <key>          returns the current value of <key>
  yodata config set <key> <value>  updates the current value of <key> to <value>
  yodata config show               print all settings

Options:
  --help        Show help                                              [boolean]
  --version     Show version number                                    [boolean]
  --output, -o  format command output                          [default: "yaml"]
```
